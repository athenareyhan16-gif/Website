About Company

Kuro Games merupakan perusahaan game developer asal China yang berdiri sejak tahun 2014. Perusahaan ini dikenal karena mengembangkan game action RPG berkualitas tinggi, salah satunya adalah Wuthering Waves.

Dengan teknologi modern gaming engine, Kuro Games berhasil menghadirkan grafik berkualitas tinggi dan combat system cepat hingga 95% lebih responsif. Sistem dunia futuristik dalam game mereka menggunakan berbagai elemen, termasuk energi H2O, api, dan listrik.

Company Vision :

  • Menjadi perusahaan game global terbaik.
  • Menciptakan pengalaman bermain berkualitas tinggi.

Company Mission :

  1. Mengembangkan game action RPG modern.
  2. Menggunakan teknologi visual terbaru.
  3. Memberikan pengalaman gaming terbaik untuk player.
